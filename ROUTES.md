# Маршруты (URL-пути) проекта

## Основные страницы

| Название | Маршрут |
|----------|---------|
| Главная страница | `/index/` |
| Каталог товаров | `/list/` |
| Карточка товара | `/product/<id>/` |
| Список товаров | `/product/` |
