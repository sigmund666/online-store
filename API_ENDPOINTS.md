# API Endpoints

**Базовый URL:** `http://127.0.0.1:8000/api/`

---

## Товары

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/products/` | Список всех товаров |
| GET | `/products/{id}/` | Карточка товара |
| GET | `/categories/` | Список категорий |

---

## Корзина

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/cart/` | Добавить товар |

**Запрос:**
```json
{
    "product_id": 1,
    "quantity": 2
}
```
