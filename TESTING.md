## Метод GET

### 1. Список всех товаров
- **Адрес:** `http://127.0.0.1:8000/api/products/`
- **Ожидаемый результат:** JSON-массив с товарами
- **Результат:** работает

### 2. Карточка товара (по ID)
- **Адрес:** `http://127.0.0.1:8000/api/products/1/`
- **Ожидаемый результат:** JSON одного товара
- **Результат:** работает

### 3. Список категорий
- **Адрес:** `http://127.0.0.1:8000/api/categories/`
- **Ожидаемый результат:** JSON-массив категорий
- **Результат:** работает

## Метод POST

### 1. Добавление товара в корзину
- **Адрес:** `http://127.0.0.1:8000/api/cart/`
- **Ожидаемый результат:** В ответе есть поля: id, product_id, quantity, unit_price
- **Тело запроса:**
```
{
    "product_id": 1,
    "quantity": 2
}
```
- **Результат:** работает

### 2. Оформление заказа
- **Адрес:** `http://127.0.0.1:8000/api/orders/`
- **Ожидаемый результат:** В ответе есть поля: id, status, total_amount, items
- **Тело запроса:**
```
{
    "full_name": "Иванов Иван Иванович",
    "phone": "+375291234567",
    "address": "Минск, ул. Ленина, д. 1, кв. 10"
}
```
- **Результат:** работает
